# 별 수가르기 — 수 가르기·모으기 학습 앱

초등 1학년 아이를 위한 **수 가르기·모으기**(number bond) 앱입니다.
태블릿에서 **직접 풀고**, A4 문제지로 **출력**해서 종이로도 풀 수 있어요. (별 사냥 스도쿠와 같은 "별" 시리즈)

이 폴더의 파일들은 모두 **같은 폴더 안에** 함께 있어야 합니다.
- index.html  (앱 본체)
- manifest.json  (앱 정보)
- sw.js  (오프라인 실행용)
- icon-192.png / icon-512.png / icon-maskable-512.png  (앱 아이콘)

## A. 가장 쉬운 방법 — GitHub Pages (무료, HTTPS)

1. github.com 에서 새 저장소(repository)를 만듭니다. 예: `byeol-sugarugi` (Public)
2. 이 폴더의 파일들을 저장소에 올립니다. ("Add file → Upload files"로 끌어다 놓아도 됩니다)
3. 저장소 → Settings → Pages → Source를 **Deploy from a branch**, Branch는 **main / (root)** 로 두고 Save.
4. 1~2분 뒤 `https://<내아이디>.github.io/byeol-sugarugi/` 주소가 생깁니다.

## B. 갤럭시탭에 설치하기

1. 갤럭시탭 **크롬**에서 위 주소를 엽니다.
2. 우측 상단 메뉴(⋮) → **"앱 설치"** 또는 **"홈 화면에 추가"** 를 누릅니다.
3. 홈 화면에 별 아이콘이 생기고, 누르면 주소창 없이 전체화면으로 실행됩니다.
4. 가로로 돌리면 왼쪽 사이드바 + 오른쪽 본문으로 보기 편하게 배치됩니다.
   (자세한 설치/문제해결은 `GALAXY_TAB_설치가이드.md` 참고)

## 사용 방법

### 🧮 풀기 (아이가 직접)
- **단계**: `10까지` → `20까지` → `30까지` 순으로 범위가 점점 커집니다. (10까지가 2~10을 포함하므로 작은 수 연습도 됩니다. 더 늘리려면 `STAGES`에 한 줄 추가)
  한 단계에서 정해진 문제 수(기본 20)를 풀면 다음 단계가 열립니다.
- **유형**(사이드바에서 바로 선택): `가르기` / `모으기` / `섞기`.
- 큰 □ 칸이 빈칸이에요. 아래 **0~9 키패드로 답을 눌러 쓴 뒤 "입력"** 을 누르면 채점됩니다. (두 자리 답은 두 번 눌러요. "지우기"로 한 자리씩 지우거나, 빈 네모를 눌러 처음부터 다시 쓸 수 있어요)
  맞으면 별이 쏟아지고 다음 문제로, 틀리면 살짝 흔들리고 다시 풀 수 있어요.
- **오늘의 미션**(기본 하루 10문제)을 끝내면 달력에 도장(★)이 찍히고 연속 일수(🔥)가 올라갑니다.

### 🖨️ 출력 (부모가 종이로)
- **유형**(가르기 / 모으기 / 섞기), **단계**, **쪽수**(1~10), **정답지 포함**을 고르고
  **"미리보기 만들기"** → **"인쇄 / PDF 저장"** 을 누릅니다.
- 한 장에 **9문제**가 들어가고, 매번 **새 문제**가 만들어집니다. (무한정 출력)

### 📅 기록
- 도장 달력, 현재/최고 연속, 푼 문제 수, 총점을 봅니다.

### 🔐 부모 모드 (난이도 조절)
- **제목("별 ⭐ 수가르기")을 길게 누르면** 부모 모드가 열립니다. (처음엔 비밀번호 만들기)
- 단계 잠금 해제(여기까지 열어주기), 오늘의 미션 수·단계 통과 문제 수 조절, 비밀번호 변경/초기화.
- **기록 초기화 (점수·별 0으로)**: 점수·별·도장·단계를 모두 0으로 되돌립니다. (실수 방지로 **두 번** 눌러 확인. 비밀번호·설정은 유지) — 부모가 시험 삼아 푼 기록을 지우고 아이가 새로 시작할 때 사용.
- (유형 가르기/모으기/섞기는 **풀기 화면 사이드바**에서 바로 고릅니다.)

## 설정 바꾸기 (index.html 위쪽 상수)

- `STAGES` 배열에 한 줄 추가하면 새 단계가 생깁니다. 예) `{ key:"s20", label:"20까지", min:11, max:20 }`
- `POINTS` : 한 문제 점수 (현재 100)
- `DEFAULTS` : 처음 설정값(유형·미션 수·단계 통과 수). 이후 값은 부모 모드에서 조절 가능.

※ 파일을 고쳐 다시 올린 뒤 변경이 바로 안 보이면, `sw.js`의 `byeolsugarugi-v1`을 `v2`로 바꿔 올리고 앱을 한 번 닫았다 켜세요.
