## 1. 문제 생성 — `makeProblem`
- 단위 `{ W, A, B, style:'split'|'merge', blank:'top'|'left'|'right' }`, 항상 `A+B===W`, 부분 ≥1(0 미사용).
- `split`(가르기): 위 W 주고 아래 한 칸 비움 / `merge`(모으기): 아래 두 수 주고 위(합) 비움. `shuffle`: 가르기 좌/우 빈칸 랜덤.
- 유형은 `save.settings.type`(`split`|`merge`|`mix`) → `optsForType()`로 생성 옵션 변환. 직전 문제와 동일 회피.

## 2. 다이어그램 — `bondSVG` (전부 인라인 SVG)
- play/print/answer 3모드. 박스+화살표(마커)+숫자 모두 SVG → 인쇄 시 벡터로 선명.
- play: 빈칸은 점멸 강조·입력값 표시, 정답 초록 / 오답 빨강. 빈 네모 탭 → 입력 초기화.

## 3. 풀기 모드
- **0~9 키패드**(`지우기 | 0 | 입력`, 12키, 다이어그램 **오른쪽**에 배치). 숫자 누른 뒤 **‘입력’으로 제출**(명시적 채점). 두 자리 답은 두 번. 오답 → 흔들림·재시도.
- **유형 선택**(가르기/모으기/섞기)을 사이드바에서 직접.
- **단계 사다리** `10까지 → 20까지 → 30까지`(min 2, max만 ↑). `levelTarget`(기본 20) 달성 시 다음 해금. 한 줄 추가로 확장(예 `{key:"s50",label:"50까지",min:2,max:50}`).
- 일일 미션(`dailyTarget` 기본 10) 완료 시 도장+연속(streak), 마일스톤 축하 모달.

## 4. 출력 모드
- 유형·단계(또는 섞기)·쪽수(1~10)·정답지 포함 선택 → "미리보기 만들기" → `window.print()`.
- A4 세로 한 장 3×3 = 9문제. 헤더(별 수가르기 + 월/일/이름 + 페이지) + 안내문 + 자체 푸터. 매번 새 문제(무한).
- `@media print`로 UI 숨기고 `#printArea`만, 페이지마다 분리. 디자인 **자체 제작**(외부 브랜딩 복제 X).

## 5. 부모 모드
- 제목 길게 누르기(`bindLongPress`) → 비번(미설정 시 생성, `btoa` 난독화).
- 기능: 단계 **여기까지 열어주기**, 오늘의 미션 수·단계 통과 문제 수 스텝퍼, **기록 초기화**(`resetProgress`, **2탭 확인** → 점수·별·도장·연속·maxLevel 0, **비번·설정 유지**, 초기화 후 부모모드 끄고 첫 단계로), 비번 변경/초기화.
- 부모 모드에서 푼 문제는 기록에 반영 안 됨.
